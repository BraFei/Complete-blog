# new-myblog
基于python的django框架搭建的一个博客网站。
## 开发环境
- Python3.6
- Django2.0

## 运行项目
1. 下载<br>
	`git clone  https://github.com/BraFei/blog-new-style.git`
2. 进入项目<br>
	`cd mysite`
3. 运行项目<br>
	`python manage.py runserver`
